# CS-255

The DriverPass project was about designing a system for a company that helps students prepare for their DMV driving test. The client wanted a web-based system where students could take practice tests, schedule driving lessons, and access training materials. The system also needed to support instructors and staff by allowing them to manage lessons, assign vehicles, and track student progress .

One thing I think I did well was clearly organizing the system requirements and making sure everything connected back to what the client wanted. I focused on making sure the system supported both the learning side, like practice tests, and the scheduling side, like booking lessons. I also think my diagrams improved over time, especially once I understood how to show the flow of actions and interactions between parts of the system.

If I could go back and revise one part of my work, it would probably be my use case diagram. At first, I didn’t fully understand how to correctly show relationships like include and extend, so I would clean that up and make the connections clearer. I would also try to improve the layout so it looks less crowded and easier to follow.

When I interpreted the user’s needs, I focused on what each type of user needed to do in the system. For example, students needed to schedule lessons and take practice tests, instructors needed to record feedback, and administrators needed to manage users and generate reports. Breaking it down by user made it easier to design a system that actually supports real tasks. It’s important to consider user needs because if the system doesn’t match how people use it, it won’t be effective no matter how well it’s built.

When I approach designing software now, I try to understand the problem first before jumping into diagrams. I’ve learned that starting with requirements and then building out the design step by step makes everything more manageable. In the future, I would keep using diagrams like activity and sequence diagrams to help visualize how the system works and make sure everything is connected properly.
